## Quick Start

```bash
# Install with Foundry
forge install luxfi/standard

# Or add to remappings
echo "@luxfi/standard/=lib/standard/src/" >> remappings.txt
```

## Architecture

The standard library follows these principles:

- **Composable**: Contracts work together seamlessly
- **Orthogonal**: Each contract does one thing well
- **Secure**: Audited and battle-tested implementations
- **Post-Quantum Ready**: Lamport signatures and threshold cryptography
