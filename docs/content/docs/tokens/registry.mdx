## Token Naming Convention

| Prefix | Meaning | Example |
|--------|---------|---------|
| `L*` | Bridge token on Lux | LETH (Ethereum on Lux) |
| `Z*` | Bridge token on Zoo | ZETH (Ethereum on Zoo) |
| `s*` | Synthetic token | sETH (Synthetic ETH) |
| `W*` | Wrapped native | WLUX, WZOO |

## Contract Implementations

All bridge tokens (L* and Z* prefixes) use the [ERC20B](/docs/tokens/erc20b) implementation:

```solidity
import "@luxfi/standard/src/tokens/ERC20B.sol";

// LETH deployment
contract LuxETH is ERC20B {
    constructor() ERC20B("Lux ETH", "LETH") {}
}
```
